Second International Workshop on Composable Data Management Systems, 2023

Workshop Venue: Vancouver, Canada - Co-located with VLDB 2023

Workshop Date: 28th August 2023

Presentations


Keynote Talks

  • Horizons of Composability by Orri Erling, Software Engineer, Meta Inc.
  • Unbundling of the DBMS stack by Mosha Pasumansky, CTO, Firebolt & Benjamin Wagner, Engineering Manager, Firebolt
  • Hybrid Query Execution; What is a database client, anyway? by Jordan Tigani, Co-Founder & CEO, MotherDuck
  • Taking Postgres into the 21st Century by Nikita Shamgunov, CEO, Neon Database

Invited Talks

  • Innovating with storage formats to push the limits in a hybrid database by Eugene Kogan, Software Architect, SingleStore
  • Holistic Extensibility for Integrated Data Analysis Pipelines in DAPHNE by Patrick Damme, Postdoctoral Researcher, Technische Universität Berlin (PPTX)

Short Paper Presentations

  • Relation-Based In-Database Stream Processingy by Christian Winter (Technical University of Munich), Thomas Neumann (Technical University of Munich), Alfons Kemper (Technical University of Munich)
  • Building a serverless Data Lakehouse from spare parts by Jacopo Tagliabue (NYU and Bauplan), Ciro Greco (Bauplan), Luca Bigon (Bauplan)
  • The Gluten Open-Source Software project - modernizing Java-based query engines for the Lakehouse era by Akash Shankaran (Intel), George Gu (Intel), Weiting Chen (Intel), Binwei Yang (Intel), Chidamber Kulkarni (Intel), Mark Rambacher (Intel), Nesime Tatbul (Intel), David E. Cohen (Intel) (PPTX)
  • E-Scan: Consuming Contextual Data with Model Plugins by Viktor Sanca (EPFL), Anastasia Ailamaki (EPFL & Google)
  • Wisent: An In-Memory Serialization Format for Leafy Trees by Hubert Mohr-Daurat (Imperial College London), Holger Pirk (Imperial College London)

Lightning Talks

  • Towards Efficient and Secure UDF Execution with BabelfishLib by Philipp Grulich, Technische Universität Berlin
  • Composable Gateway for Databases by Aditya Jalan, Meta
  • Polygraph - An Arrow Format for Large Apache Calcite Query Plans and Multi-Language Data Systems by Adam Kennedy, Voltron Data
  • Technics in Accelerating Query Processing on GPU by Jimmy Lu, Meta
  • Using Multiple Composable, Hardware-Accelerate Executors by Felipe Aramburu, Voltron Data Inc.
  • The Case for Decoupled Transaction Managers by Hiroyuki Yamada, Scalar, Inc.
